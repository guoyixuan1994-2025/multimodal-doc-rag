# 政企文档智能检索系统说明

政企文档智能检索系统用于帮助项目经理、售前人员和运维人员快速查询项目资料。
系统不直接替代人工决策，而是基于文档资料提供可溯源的问答能力。

## 核心流程

系统流程包括文档解析、文本清洗、chunk 切分、embedding 入库、混合检索、reranker 精排和大模型生成。
当资料不足时，系统应该拒答，而不是编造答案。

## 文档解析

PDF 文档优先使用 MinerU 解析，提取正文、标题、页码、表格和图片区域。
Markdown 文档适合按标题规则分块。
TXT 文档适合使用递归分块。

## 检索策略

BM25 适合匹配专有名词、产品名、指标名和政策编号。
向量检索适合匹配语义相近但字面不同的问题。
混合检索可以同时利用关键词信号和语义信号。

## 质量评估

系统需要评估检索质量和回答质量。
检索质量可以使用 Hit Rate、Precision@K、Recall@K 和 MRR。
回答质量可以使用 RAGAS 的 answer relevancy、faithfulness、context precision 和 context recall。
