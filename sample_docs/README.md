# Sample Documents

This directory contains lightweight files for demonstrating ingestion, OCR, image understanding, and retrieval.

Large third-party papers are intentionally not committed. To test with public research papers, download them from their official sources and upload them through the UI:

- [BERT](https://arxiv.org/abs/1810.04805)
- [Chain-of-Thought Prompting](https://arxiv.org/abs/2201.11903)

Before publishing additional sample documents, confirm that they do not contain private data and that redistribution is permitted.
